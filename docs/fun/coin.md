# Coin Flips

The `/coin` command lets you flip a coin. That's it.

* `call` choose if you think you'll get heads or get tails

<!-- panels:start -->
<!-- panels:title -->
Example:
<!-- div:left-panel -->
```invocation
/coin call:heads
```
<!-- div:right-panel -->
<@user> called *heads* and flipped a coin. They got **tails**.
<!-- panels:end -->
